PGN load

Starts POST /Analyser/LoadGames and polls GET /Analyser/LoadGamesProgress. Cancel requests POST /Analyser/CancelLoad.

Analytics metrics

Discovery: GET /api/analytics/metrics. Execute: POST /api/analytics/metrics/execute.

Optional side filters for metrics that operate on games or moves.

This corpus trend averages White-side and Black-side material in the selected game set. It is not a player comparison.

Player comparison uses full move number. Full move N maps internally to PlyIndex = (N * 2) - 1, after Black's Nth move.
